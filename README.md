# Murphy's Laws Random Quote API

## Getting Started

```sh
python3 -m venv .venv

. .venv/bin/activate
```

## Install dependencies

```sh
pip install -r requirements.txt
```

Or

```sh
yarn setup
```

## Start app

```sh
flask --app app.py --debug run
```

Or

```sh
yarn start
```

## Tests

```sh
pytest
```
