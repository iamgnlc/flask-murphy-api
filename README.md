# Murphy's Laws Random Quote API

![ci-cd](https://github.com/iamgnlc/flask-murphy-api/actions/workflows/ci-cd.yml/badge.svg)

Returns random Murphy's Law quote(s).

## Getting Started

```sh
python3 -m venv .venv

. .venv/bin/activate
```

## Install dependencies

```sh
make install
```

## Run app in dev

```sh
make dev
```

## Run app in prod

```sh
make start
```

**NOTE:** It uses `waitress` server.

## Tests

```sh
make test
```
